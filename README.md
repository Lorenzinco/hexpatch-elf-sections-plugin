# Hexpatch elf section plugin

This is a plugin for [Hex-patch](https://github.com/Etto48/HexPatch) that adds a command to the command list.

To use the plugin, move it to the plugins folder or launch hexpatch with the `-p` argument and specify the folder in which the plugin is.

When using hexpatch press `space` and look for the command `elf_sections`.
